❤牛顿迭代法

通过迭代来向目标值进行逼近，详解如下图：

重点：

  • x1 = (x + a / x) / 2
  • 用r > x / r代替r^2 > x可以防止溢出
  • long r = x;
    当r=2^31-1时，r + x / r会溢出哦！
class Solution {
    public int mySqrt(int x) {
        if (x <= 1) return x;
        long r = x;
        while (r > x / r) { // r^2 > x 防止溢出
            r = (r + x / r) / 2;
        }
        return (int)r;
    }
}
